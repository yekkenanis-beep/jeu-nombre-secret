#  Jeu du Nombre Secret

# Description
Application web interactive développée avec **Streamlit** où l'utilisateur doit deviner un nombre secret généré aléatoirement entre 1 et 100.

# Fonctionnalités
- Saisie d'une estimation
- Indicateurs "plus grand" / "plus petit"
- Compteur de tentatives
- Historique des essais
- Animation de ballons en cas de victoire

# Technologies utilisées
- Python 3
- Streamlit

# Installation et exécution
```bash
pip install streamlit
streamlit run app.py
